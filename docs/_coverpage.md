# mui

> Write once, deploy everywhere.

A cross-platform UI abstraction for Haxe. Build native apps for macOS/iOS, Windows, and terminal from a single codebase.

- Unified API across `sui` (SwiftUI), `wui` (WinUI), and `cui` (TUI)
- Zero runtime overhead via compile-time backend selection
- Type-safe bindings with `@:from` abstracts
- Macro-based `ForEach` that works across codegen and runtime backends
